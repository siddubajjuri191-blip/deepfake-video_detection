==============================================

===============

-----------------------------

KEY FEATURES:
=============

1. EXPLAINABLE PREDICTIONS
   - Shows WHY the model classified a video as fake
   - Highlights suspicious facial regions with red bounding boxes
   - Per-frame confidence tracking across entire video

2. GRAD-CAM VISUALIZATION
   - Heatmap overlay on original frames
   - Color-coded intensity (red=high, yellow=medium, blue=low)
   - Toggleable original/heatmap comparison view

3. ARTIFACT SCORING
   - Facial Landmark Anomaly Score: Eye/mouth/nose positioning deviations
   - Blending Artifact Score: Face boundary synthesis imperfections
   - Temporal Inconsistency Score: Frame-to-frame variation patterns
   - Spatial Artifact Density: Overall abnormality concentration

4. FRAME-BY-FRAME ANALYSIS
   - Process 10 frames per video (configurable)
   - Per-frame predictions and confidence
   - Clickable timeline to inspect individual frames
   - Visual confidence chart showing variation across frames

5. DEMO MODE
   - No backend required to explore UI
   - Click "Run Demo" for synthetic sample analysis
   - Full feature demonstration with mock data

TO GET STARTED:
===============

1. Frontend is ready now:
   - npm run dev      # Development server
   - npm run build    # Production build (done)

2. To enable real analysis, set up backend:
   - See BACKEND_SETUP.md for detailed instructions
   - Quick: cd backend && pip install -r requirements.txt
   - Then: python manage.py runserver

3. Or use Demo Mode (no setup needed):
   - Click "Run Demo" in the upload zone

ARCHITECTURE:
==============

Frontend Flow:
1. User uploads video or clicks "Run Demo"
2. If real: Send to backend via FormData POST
3. Backend processes with Grad-CAM:
   - Extract frames at regular intervals
   - Load trained model
   - For each frame:
     * Run forward pass (get prediction)
     * Compute gradients for target class
     * Generate Grad-CAM heatmap
     * Extract bounding boxes of suspicious regions
   - Calculate aggregate scores
4. Return JSON with per-frame heatmaps and metrics
5. Frontend renders interactive visualization

Data Format (API Response):
{
  overallPrediction: "fake",
  overallConfidence: 0.87,
  frameAnalyses: [
    {
      frameIndex: 0,
      timestamp: 0.5,
      prediction: "fake",
      confidence: 0.92,
      originalImage: "base64_jpeg",
      gradcamImage: "base64_heatmap_overlay",
      suspiciousRegions: [
        { x: 0.25, y: 0.3, width: 0.15, height: 0.12, intensity: 0.95 }
      ]
    }
  ],
  explanation: {
    topReasons: ["reason1", "reason2"],
    artifactScore: 0.82,
    temporalInconsistencyScore: 0.74,
    blendingArtifactScore: 0.91,
    facialLandmarkScore: 0.88
  }
}
